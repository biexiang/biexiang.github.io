压缩

构造输入字符的优先队列

统计单个字符出现的次数为一个map[string]int，并且按照出现的次数从小到大排列。之后将这个map构造为一棵哈夫曼树。

首先优先队列是一个抽象数据类型，一般使用堆来实现，Golang中有container/heap，在PHP里标准库中有实现SplPriorityQueue。这里使用Golang做简单的包装。

Golang对于任何实现了container/heap接口的都提供优先队列的操作。
需要实现的接口有:

  • Len()int
  • Less(i,j int) bool
  • Swap(i,j int)
  • Push(x interface{})
  • Pop() interface{}

需要注意的是，Golang中类型转换interface到其他数据类型方法为：

    var x interface{} = "mmp"
    val,ok := x.(string)        //转换失败panic，not ok

还有Less实现的判断和Pop的顺序需要是一致的。

首先结构定义：

    //结构定义
    type Node struct {
        value  string //字符 或者asc、unicode值
        weight int    //权重
        left   *Node
        right  *Node
    }
    type Nodes []*Node

下面是container/heap接口的实现：

    //实现5个接口
    func (n Nodes) Len() int {
        return len(n)
    }

    func (n Nodes) Less(i, j int) bool {
        return n[i].weight < n[j].weight
    }

    func (n Nodes) Swap(i, j int) {
        n[i], n[j] = n[j], n[i]
    }

    func (n *Nodes) Push(x interface{}) {
        add := x.(*Node)
        *n = append(*n, add)
    }

    func (n *Nodes) Pop() interface{} {
        data := *n
        first := data[0]
        *n = data[1:]
        return first
    }

构造哈夫曼树

首先对Map数据使用优先队列最小堆化。再依次Pop两个元素从下到上形成树，直到队列里只有一个元素。就是根节点。

    //Huffman树构建
    func BuildTree(x map[string]int) *Node {
        queue := &Nodes{}
        heap.Init(queue)

        for str, cnt := range x {
            tmp := &Node{
                value:  str,
                weight: cnt,
            }

            heap.Push(queue, tmp)
        }

        //优先队列OK

        //构建树
        for queue.Len() > 1 {
            left := queue.Pop().(*Node)
            right := queue.Pop().(*Node)

            tmp := &Node{
                value:  "",
                weight: left.weight + right.weight,
                left:   left,
                right:  right,
            }
            queue.Push(tmp)
        }
        return queue.Pop().(*Node)
    }

构造哈夫曼编码表格

从根节点开始遍历，value不为空的存储叶子节点的01串。

    //遍历树获取哈夫曼编码
    func GetHuffmanCode(node *Node, box map[string]string, prefix string) {
        if node.value != "" {
            //叶节点
            box[node.value] = prefix
        } else {
            lprefix := prefix + "0"
            GetHuffmanCode(node.left, box, lprefix)
            rprefix := prefix + "1"
            GetHuffmanCode(node.right, box, rprefix)
        }
    }

输出存储哈夫曼编码表格和对应的压缩数据01串

    //获取压缩后的01串
    func GetHuffman(before string, table map[string]string) string {
        var resp string = ""
        for i := 0; i < len(before); i++ {
            resp += table[string(before[i])]
        }
        return resp
    }

解压

在压缩存储时编码表应该也作为一部分进行压缩，解压的时候对于压缩后的01串进行编码表的匹配，匹配到了就替换为字符串。
