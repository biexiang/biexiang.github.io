Golang实现排列组合

2017-11-06

上次面试的题目。

全排列

如下图所示，首先获取第一位不同的字串，再依次递归下去，直到index = length - 1。输出字符串。 

    /**
     *  +全排列
     *      ++递归排列
     */
    func rangeAll(slice []string, start int) {
        size := len(slice)
        if start == size-1 {
            //如果已经是最后位置了，直接将数组数据合并输出
            fmt.Println(strings.Join(slice, ""))
        }

        for i := start; i < size; i++ {

            //  1、abc输出abc，i=start时，输出自己
            //  2、如果i和start的值相同没有必要交换，交换后输出的是i=start时输出的内容，是重复的内容
            if i == start || slice[i] != slice[start] {
                //交换当前这个与后面的位置
                slice[i], slice[start] = slice[start], slice[i]
                //递归处理索引+1
                rangeAll(slice, start+1)
                //换回来，因为是递归，如果不换回来会影响后面的操作，并且出现重复
                slice[i], slice[start] = slice[start], slice[i]
            }

        }

全组合

abc，长度为3，每一位是否出现显示为1/0，abc即111，ab即110，c即001，若给一个字符串求出全组合，即2^n - 1个，因为000(不出现一个字符)有毛病。

    /**
     *  +全组合
     *      +位图法
     */
    func combination(slice []string) {
        str := strings.Join(slice, "")
        size := len(str)
        max := 1 << uint64(size)

        //至少出现一个字符
        var word string
        for i := 1; i < max; i++ {

            word = ""
            for j := 0; j < size; j++ {
                tmp := i
                if tmp&(1<<uint64(j)) != 0 {
                    word += string(str[j])
                }
            }
            fmt.Println(word)
        }

    }
